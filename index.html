Hola, soy Oscar :), un FullStack Javascript Developer entusiasmado por la tecnología

Actualmente me dedico a desarrollar en tecnologías web principalmente como React.js y Node.js.

Así mismo, he desarrollado aplicaciones Front-End utilizando React js, Apollo Client para consumir APIs en GraphQL, ant Design para gráficos, Styled-Components, React Hooks, PWA, React Intl para traducción de lenguajes, React GA y html 2 canvas para la captura de imágenes.

En el lado del back-end, he desarrollado aplicaciones usando Node.js, Daemon y Rabbit.MQ para la ejecución progresiva de procesos externos, Sequelize, mySQL Workbench, ES6, bodybuilder para parseo de queries a elastic, GraphQL, Apollo Server, websockets y python para web scraping.

Finalmente, he desarrollado aplicaciones móbiles en Flutter y en React Native recientemente.
